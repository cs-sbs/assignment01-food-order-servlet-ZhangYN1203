# 截图说明

请将以下页面的运行截图放入此目录：

## 需要的截图

1. **菜单列表页面**
   - URL: http://localhost:8080/menu
   - 文件名: menu_list.png 或 menu_list.jpg

2. **创建订单页面**
   - URL: http://localhost:8080/
   - 文件名: order_create.png 或 order_create.jpg

3. **订单详情页面**
   - URL: http://localhost:8080/order/1001 (先创建订单获取ID)
   - 文件名: order_detail.png 或 order_detail.jpg

## 截图要求

- 确保截图清晰可见页面内容
- 截图应显示完整的页面布局
- 建议使用PNG格式保存

## 添加截图后

添加截图后，使用以下命令提交：
```bash
git add src/main/resources/
git commit -m "添加运行截图"
git push origin main
```
